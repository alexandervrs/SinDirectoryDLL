# SinDirectoryDLL

An old Directory DLL extension for Game Maker 8 games

SinDirectory DLL accesses special folders in Game Maker 8 & provides a few folder operations like:

```
Create
Rename
Delete (only empty folders)
Check if folder exists
```

Special folders the DLL can return to the game:

```
desktop
roaming_appdata
local_appdata
documents
music
pictures
video
profile
sendto
startmenu
startup
templates
burn
fonts
resources
programfiles
programfilesx86
programfilescommon
system
systemx86
windows
programs
temp
currentworking
```
